## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/Fud0o0/SyndraShell.git
cd SyndraShell

# Run installation script
chmod +x install.sh
./install.sh

# Configure your settings
cp config.example.json config/config.json

# Launch SyndraShell
./syndrashell.sh
```
