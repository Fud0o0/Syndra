# Default Wallpaper

Place your default wallpaper video here as `default.mp4`.

This file will be automatically copied to `~/Pictures/Wallpapers/default.mp4` during installation.

## Requirements
- Format: MP4, MKV, AVI, or any format supported by mpv
- Recommended: MP4 with H.264 codec for best compatibility
- Resolution: 1920x1080 or higher recommended

The wallpaper will be displayed using mpvpaper with loop and no-audio options.
