# Syndra Installation Scripts

This directory contains the installation scripts for different team configurations.

## Available Scripts

- `red` - Red Team (Offensive Security) - 15 GB required
- `blue` - Blue Team (Defensive Security) - 12 GB required  
- `purple` - Purple Team (Full Spectrum) - 25 GB required
- `root` - Root Me (CTF Focused) - 18 GB required

## Usage

```bash
# Red Team
curl -L get.syndra.me/red | sh

# Blue Team
curl -L get.syndra.me/blue | sh

# Purple Team
curl -L get.syndra.me/purple | sh

# Root Me
curl -L get.syndra.me/root | sh
```

All scripts require Arch Linux with Hyprland.
